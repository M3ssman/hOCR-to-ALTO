# hOCR-to-ALTO
Convert between Tesseract hOCR and ALTO XML 2.0/2.1 using XSL stylesheets

See [ocr-fileformat](https://github.com/UB-Mannheim/ocr-fileformat) for an
interface to using these stylesheets.
